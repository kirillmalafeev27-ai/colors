# Deutsch lesen — учимся читать по-немецки 👧

Маленькое веб-приложение для ребёнка, который ещё не знает букв. Сверху —
цветные **кружки** (буквы) и **квадратики** (буквосочетания) с озвучкой.
Ниже — такие же **пустые** фигуры. Ещё ниже — **палитра** из цветов и букв.
Задача: нажать пустую фигуру, услышать звук и подобрать ей правильный цвет и
букву из палитры.

Озвучка — через **ElevenLabs**. Каждый звук запрашивается у API **один раз** и
кэшируется (на сервере и в браузере), чтобы не тратить лимиты впустую.

## Что нужно
1. Аккаунт на **ElevenLabs** и API-ключ (Profile → API Keys).
   (По желанию — Voice ID понравившегося голоса из раздела Voices.)
2. Аккаунт на **Render** и репозиторий на GitHub с этими файлами.

## Деплой на Render (5 минут)
1. Залейте папку проекта в репозиторий на GitHub.
2. На Render: **New → Web Service** → подключите репозиторий.
3. Настройки:
   - **Runtime:** Node
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
4. Вкладка **Environment** → добавьте переменную:
   - `ELEVENLABS_API_KEY` = ваш ключ
   - (опционально) `ELEVENLABS_VOICE_ID`, `ELEVENLABS_MODEL_ID`, `ELEVENLABS_LANG`
5. **Create Web Service** → дождитесь деплоя → откройте выданный URL.

> На бесплатном тарифе сервис «засыпает» без активности — первый запрос после
> простоя может открываться ~30 секунд. Это норма.

## Локальный запуск (по желанию)
```bash
npm install
ELEVENLABS_API_KEY=ваш_ключ npm start
# откройте http://localhost:3000
```

## Как менять уроки и звуки
Откройте `public/index.html`, найдите массив `LESSONS`.
- `L('a','a')` — буква (кружок).
- `C('Sch','sch')` — буквосочетание (квадрат).
- Второй аргумент — «ключ цвета»: одинаковые буквы во всех словах будут
  одного цвета. Цвета заданы в `GRAPHEME_COLOR` выше.
- Третий (необязательный) аргумент — что отправлять в озвучку, если звук по
  умолчанию звучит не так, как нужно. Пример: `L('ß','ß','ss')`.

Слова можно разбивать на части (массивы внутри `words`) — между ними будет
пробел, как во фразе `Wie heißt du?`.

## Безопасность
API-ключ хранится только на сервере (переменная окружения) и **никогда** не
передаётся в браузер. Браузер общается только с вашим эндпоинтом `/api/tts`.
